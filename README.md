# Portfolio Website (Next.js • MongoDB • Cloudinary)

This project is a full-stack personal portfolio built with **Next.js**, **MongoDB**, and **Cloudinary**. It serves as a dynamic platform to showcase projects, blog posts, skills, and professional experience while providing an admin dashboard for managing content with ease.

## Key Features

* **Dynamic Portfolio Management**
  Add, update, or delete projects, blogs, and media through an integrated admin panel.

* **Next.js Frontend & API Routes**
  Utilizes server-side rendering (SSR) and static generation (SSG) for fast performance, SEO optimization, and improved user experience.

* **Cloudinary Media Storage**
  Seamless image and asset uploads with automatic optimization, transformations, and fast CDN delivery.

* **MongoDB Database**
  Stores all portfolio data (projects, blog posts, user info, images) with a scalable and secure schema design.

* **Authentication & Authorization**
  Secure admin login to manage content, protect routes, and control access to backend APIs.

* **Responsive Modern UI**
  Clean, mobile-friendly interface built using reusable components and modern styling.

* **Optimized Performance**
  Implemented caching, lazy loading, and optimized queries for better loading speed and scalability.

## Tech Stack

* **Framework:** Next.js 14+
* **Frontend:** React, Tailwind CSS
* **Backend:** Next.js API Routes (Node.js/Express-style logic)
* **Database:** MongoDB (Mongoose ODM)
* **Media Storage:** Cloudinary
* **Authentication:** NextAuth / JWT
* **Deployment:** Vercel (recommended), MongoDB Atlas, Cloudinary

## Purpose

This portfolio is designed to present professional experience, highlight major projects, publish blog content, and provide an easily maintainable platform that can scale as new skills and achievements are added. It also demonstrates real-world full-stack development capabilities across authentication, media handling, database operations, and modern UI engineering.
